# Golf Practice App — Feature Index

Detailed feature specs live in `features/`. Read the relevant file(s) before making changes.

| File | Covers |
|---|---|
| `features/home.md` | Home tab — template selection, active block view |
| `features/history.md` | History tab — block list, block detail, completion summary |
| `features/progress.md` | Progress tab — per-drill charts, trend line |
| `features/drills.md` | Drills tab — drill library, templates, scoring wizard, single drill scorer |
| `features/shared.md` | Navigation, data/business logic, UI patterns (cross-cutting) |
